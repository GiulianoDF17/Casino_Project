PURPOSE OF PROJECT: store, view, and create recipes
VERSION or DATE: Version 0.0.1
HOW TO START THIS PROJECT: Right click on Recipe and click (void main (String[] args)), and then you may pick any options.
AUTHORS: Guliano Di Fulvio and Daniel Corcuera Lodico
USER INSTRUCTIONS: Once you're done adding all the ingredients in the recipe please enter 'done', to finish adding.

How to run outside of BlueJ:

1. Open command prompt (Windows + R) and typing cmd.
2. Install java (if not installed) via this link https://www.oracle.com/java/technologies/downloads/#jdk25-windows .
3. Change the directory to the path of the project using (cd directory).
4. Compile the program by doing (javac Recipe.java).
5. Run the program by doing (java Recipe).
6. Done! ENJOY!!
